**Welcome to your Voter Alliance project** 

**About**

View and Edit  your app on [Voter Alliance.com](http://Voter Alliance.com) 

This project contains everything you need to run your app locally.

**Edit the code in your local development environment**

Any change pushed to the repo will also be reflected in the Voter Alliance Builder.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_APP_APP_ID=your_app_id
VITE_APP_APP_BASE_URL=your_backend_url

e.g.
VITE_APP_APP_ID=cbef744a8545c389ef439ea6
VITE_APP_APP_BASE_URL=https://my-to-do-list-81bfaad7.voter-alliance.app
```

Run the app: `npm run dev`

**Publish your changes**

Open [Voter Alliance.com](http://Voter Alliance.com) and click on Publish.
